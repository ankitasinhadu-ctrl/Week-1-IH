# Product Listing Generator using ChatGPT API

This project uses OpenAI's API to generate product listings from images.

## Features
- Image analysis using GPT-4o
- Automatic product description generation
- JSON output format

## Files
- product_listing_generator.py → main script
- generated_listings.json → output
- LAB API Calling to ChatGPT Lab.ipynb → notebook version

## How to run
1. Add your API key in env file
2. Run the script:
   python product_listing_generator.py
